## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/sneha-yadav786/Learning-flask.git
```

2. Navigate to the folder

```bash
cd Learning-flask
```

3. Install dependencies

```bash
pip install flask
```

4. Run the app

```bash
python app.py
```

5. Open in browser

```
http://127.0.0.1:5000/
```
